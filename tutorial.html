How to Use the Tool

This page explains the basic workflow for using the automated scanner in a safe and educational context.

Step 1: Choose the target environment

Make sure the system or network being tested is one you are authorised to assess.

Step 2: Prepare the tool

Confirm that the file is in the correct directory and the environment is ready.

Step 3: Run the command

python scanner.py 192.168.1.10

Step 4: Review the results

After the scan finishes, review the displayed output and document the findings.

Step 5: Apply responsible practice

Use all results for learning, reporting, or authorised defensive purposes only.

Legal and ethical use is essential. Never test systems without permission.
